# CodeAlpha Image Gallery

A responsive image gallery built using HTML, CSS, and JavaScript as part of the CodeAlpha Frontend Development Internship.

## Features

- Category-based image filtering
- Image effects:
  - Original
  - Grayscale
  - Sepia
  - Blur
  - Brightness
- Lightbox image preview
- Next/Previous navigation
- Responsive grid layout

## Technologies Used

- HTML5
- CSS3
- JavaScript
